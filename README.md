# ml-model-versioning
# ML Model Versioning Project

## Project Goal
This project demonstrates how to use GitHub for version control in a machine learning workflow, including dataset management, model training, and model versioning.

## Dataset
We use the Iris dataset, a classic dataset for classification tasks. It contains:
- 150 samples
- 4 features: sepal length, sepal width, petal length, petal width
- 3 target classes

## ML Algorithm
Logistic Regression is used for classification.

## Versioning
- Code versioning: Git & GitHub
- Model versioning: Git tags and saved model artifacts
